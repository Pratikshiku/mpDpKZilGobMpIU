## Inventory Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0b99115b6c404339b678177df77e927c/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Center
5. Product Information
6. Product Reservation Information
7. Purchase Receipt Management
8. Purchase Receipt Details
9. Customer Information
10. Announcement Information


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Warehouse Management, Supplier Type Management, Supplier Credit Rating Name Management, Product Type Management, Announcement Type Management)
5. Supplier Management
6. Product Management, Product Information Reports, Product Reservation Management
7. Purchase Receipt Management, Purchase Receipt Details Management
8. Customer Management
9. Announcement Information Management
10. Employee Management
